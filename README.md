## Project Overview

This project performs sentiment analysis on Amazon customer reviews using Python and Natural Language Processing (NLP). The system classifies reviews into Positive, Negative, and Neutral categories and generates visualizations for data insights.

## Features

- Sentiment Classification
- Polarity Analysis
- Data Visualization
- Word Cloud Generation
- CSV Report Generation

## Technologies Used

- Python
- Pandas
- TextBlob
- Matplotlib
- WordCloud

## Dataset

Amazon Reviews Dataset (21,214 Reviews)

## Outputs

- Sentiment Results CSV
- Sentiment Summary CSV
- Pie Chart
- Bar Chart
- Word Cloud
