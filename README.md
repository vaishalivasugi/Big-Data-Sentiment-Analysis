# Big Data Sentiment Analysis Using Python

## Project Overview
This project performs sentiment analysis on Amazon customer reviews using Python and Natural Language Processing (NLP).

## Features
- Sentiment Classification
- Polarity Analysis
- Data Visualization
- Word Cloud Generation
- CSV Report Generation

## Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib
- WordCloud

## Dataset
Amazon Reviews Dataset (21,214 Reviews)

## Outputs
- Sentiment Results CSV
- Sentiment Summary CSV
- Pie Chart
- Bar Chart
- Word Cloud
